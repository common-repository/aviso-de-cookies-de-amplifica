Plugin para mostrar el aviso de cookies en tu sitio web. Totalmente amigable con SEO.
== Description ==
Con el plugin de WordPress de Aviso de cookies de Amplifica podrás notificar el uso de las cookies en tu sitio web, cumpliendo así la normativa española. 

El plugin está diseñado para ser totalmente ligero, modificable en aspecto y no genera ningún tipo de código intrusivo en tu plantilla. A la hora de modificarlo, podrás elegir los mensajes, los colores y el tamaño del texto, sin preocuparte de generar código CSS in-line para no perjudicar en nada al rendimiento de tu sitio web. 

Además, es totalmente traducible con plugins de terceros como WPML o Qtranslate-x para ser utilizado y mantenido en un sitio multilenguaje.

== Installation ==
El plugin se puede instalar desde el gestor interno de WordPress, o descomprimiendo su contenido directamente en el directorio de plugins de tu instalación.

== Frequently Asked Questions == 

= El plugin me da un error de escritura =
Es posible que la instalación del plugin necesite que el fichero */plugins/adc_amplifica/css/adc_aviso_cookies_style.css* tenga permisos de escritura. 

= ¿Es un plugin gratuito? =
Si, es totalmente gratuito.
